### GitHub Issues

For bug reports and feature requests, please use GitHub Issues:

**[GitHub Issues](https://github.com/sam-fakhreddine/aws-containers/issues)**

Before creating an issue:
1. Search existing issues to avoid duplicates
2. Review the [troubleshooting guide](./docs/user-guide/troubleshooting.md)
3. Use the appropriate issue template:
   - [Bug Report](./.github/ISSUE_TEMPLATE/bug_report.yml)
   - [Feature Request](./.github/ISSUE_TEMPLATE/feature_request.yml)
   - [Documentation Issue](./.github/ISSUE_TEMPLATE/documentation.yml)

## Security Issues

**Do NOT use GitHub Issues for security vulnerabilities.**

Please report security issues according to our [Security Policy](./SECURITY.md).

## Contributing

If you'd like to contribute to the project:

1. Read the [Contributing Guide](./docs/development/contributing.md)
2. Review the [Code of Conduct](./CODE_OF_CONDUCT.md)
3. Check out [good first issues](https://github.com/sam-fakhreddine/aws-containers/labels/good%20first%20issue)

## Response Times

This is an open-source project maintained by volunteers. Response times may vary:

- **Security Issues**: Within 72 hours (see [SECURITY.md](./SECURITY.md))
- **Bug Reports**: We aim to respond within 1 week
- **Feature Requests**: We aim to respond within 2 weeks
- **General Questions**: Response time varies based on maintainer availability

Please be patient and respectful of maintainers' time.

## What to Include When Asking for Help

To help us help you more effectively, please include:

### For Extension Issues

- Extension version (from `about:addons`)
- Firefox version
- Operating system and version
- Steps to reproduce the issue
- Expected vs actual behavior
- Browser console errors (F12 â†’ Console tab)
- Screenshots if applicable

### For API Server Issues

- API server version
- Python version (`python --version`)
- Operating system and version
- Configuration (without sensitive data)
- Server logs (terminal output)
- Error messages
- Steps to reproduce

### For General Questions

- What you're trying to accomplish
- What you've already tried
- Relevant parts of your setup
- Any error messages
